# E-Commerce-WebSite
E-Commerce website written on Java using Jakarta Servlet

## Website functionality

- [x] Login / Registration
- [x] Products list
- [x] Cart
- [x] Order products
- [x] Orders history
- Admin panel
- [x] User roles
- [ ] Orders management
- [ ] Adding / Editing / Removing products
